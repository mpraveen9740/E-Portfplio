# E-Portfolio - M Praveen

A modern, professional, and interactive E-Portfolio website for a Computer Science Engineering student, AI/ML enthusiast, and Python Developer.

## 🚀 Features

- **Premium Modern UI**: Sleek default dark mode with custom neon purple/cyan accents and clean, responsive transitions to a light theme.
- **Glassmorphism Layout**: Floating card effects using modern backdrop blurs.
- **Interactive Skills Progress**: Dynamic progress bars that animate as you scroll.
- **Academic Projects**: Showcase of key projects (Python, C, and AI/ML modules) with technology tags and codebase links.
- **Professional Certifications**: Interactive grid gallery with modal zoom previews.
- **Interactive Online Resume**: Clean, professionally styled digital resume modal with a download prompt.
- **Validated Contact Form**: Complete verification checks (email formatting, message constraints) and visual status loaders.

## 🛠️ Setup and Installation

To run this portfolio website locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mpraveen9740/E-Portfplio.git
   cd E-Portfplio
   ```

2. **Serve files locally**:
   Serve the directory using Python's static HTTP server:
   ```bash
   python -m http.server 8000
   ```
   Or using node:
   ```bash
   npx serve .
   ```

3. **Open the browser**:
   Navigate to `http://localhost:8000/index.html` to view the website.
